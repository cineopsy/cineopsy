🔥 Firebase Setup

One-time setup — done forever! 🎯

  2. Create a new project — name: cineopsy
  3. In your project: click Add app → Web (</>)
  4. Register the app — copy your firebaseConfig
  5. In left menu: Firestore Database → Create database → Test mode → Enable
  6. Paste your config values in the form below
  7. Save — then re-upload the site to Netlify
  8. Done! ✅ Changes will now reflect everywhere!
